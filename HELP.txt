zist - Local ZSH history aggregation tool

Quick start:
  task build                          # Build the binary
  task test                           # Run tests
  zist collect                        # Collect from ~/.histories (default)
  zist collect ~/other/               # Collect from specific directory
  zist search docker                  # Search with fzf + preview pane
  zist search --since 2024-01-01 git  # Search with time filter
  zist --version                      # Show version

Search flags:
  --limit N                           # Max results (default: 500)
  --since DATE                        # After date (YYYY-MM-DD)
  --until DATE                        # Before date (YYYY-MM-DD)

Collect flags:
  --quiet                             # Suppress output

ZSH integration:
  zist install                        # Add Ctrl+X binding to ~/.zshrc
  zist uninstall                      # Remove integration

History sync setup:
  helpers/setup-history-sync.sh       # Set up ~/.histories/<hostname>/

Development:
  task fmt                            # Format code
  task check                          # Run all checks (fmt, vet, test)
  task deps-update                    # Update dependencies

Database:
  task db-shell                       # Open SQLite shell
  task db-backup                      # Backup database
  task db-reset                       # Delete and reset database

Installation:
  task install-user                   # Install to ~/go/bin
  task install                        # Install to /usr/local/bin (requires sudo)

Other:
  task clean                          # Clean build artifacts
  task release                        # Build release binaries
  task --list                         # List all tasks
